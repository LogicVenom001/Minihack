# Smart Meter Power Consumption Profiling

## Objective

Analyze electricity consumption data using Python, Data Cleaning, Exploratory Data Analysis (EDA), and Machine Learning.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn

## Workflow

1. Data Loading
2. Data Cleaning
3. Missing Value Handling
4. Exploratory Data Analysis
5. Data Visualization
6. Machine Learning
7. Insights Generation

## Expected Outcome

Identify consumption patterns, peak usage periods, and generate actionable insights.
