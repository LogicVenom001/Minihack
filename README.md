# Smart Meter Power Consumption Profiling

## Objective

To analyze household electricity consumption data using Python, Data Cleaning, Exploratory Data Analysis (EDA), Data Visualization, and Machine Learning techniques to identify usage patterns and generate actionable insights.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn

## Dataset

The project uses a household power consumption dataset containing electrical measurements such as active power, reactive power, voltage, current intensity, and sub-metering values.

## Project Workflow

1. Data Loading and Exploration
2. Data Cleaning and Preprocessing
3. Missing Value Handling
4. Exploratory Data Analysis (EDA)
5. Data Visualization
6. Correlation Analysis
7. Machine Learning Model Development
8. Performance Evaluation
9. Insight Generation

## Key Findings

* More than 1 million power consumption records were analyzed.
* Missing values were identified and handled appropriately.
* Global Intensity showed a very strong correlation with Global Active Power.
* Significant variations in power consumption were observed over time.
* Machine Learning was successfully used to predict power consumption patterns.

## Expected Outcome

The project helps identify electricity consumption trends, peak usage periods, influential factors affecting power usage, and provides insights that can support energy optimization and efficient resource management.
